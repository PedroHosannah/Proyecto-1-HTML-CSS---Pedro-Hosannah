PORTFOLIO PROFESIONAL - PEDRO HOSANNAH NAVAS

Archivos principales:
- index.html: página de inicio y resumen del portfolio.
- sobre_mi.html: sobre mí, formación, habilidades, idiomas, objetivos y reflexión final.
- cv.html: currículum vitae académico-profesional con QR para descargar el CV.
- portafolio.html: proyectos realizados.
- contacto.html: formulario, correo, teléfono y redes profesionales.
- styles.css: estilos del sitio, responsive y fondo visual.
- images/: carpeta con todas las imágenes del portfolio.

Para abrir el portfolio:
1. Descomprime el ZIP.
2. Abre index.html en el navegador.
3. Mantén la carpeta images junto a los archivos HTML para que todas las imágenes se carguen correctamente.
