TFG / Proyecto Intermodular ASIR 1

VECNA CORE

Proyecto principal de 1º de ASIR desarrollado como una empresa tecnológica ficticia, con una infraestructura empresarial orientada a ciberseguridad, monitorización, servicios internos y alta disponibilidad. La solución contempla Windows Server 2019, Windows 11 Pro, segmentación por VLANs, arquitectura jerárquica en estrella, políticas Zero Trust y servicios como AD DS, DNS, DHCP, FTP y WEB.

Detalle del TFG: VECNA CORE

Arquitectura de red

Diseño de una red privada 10.20.0.0/16 con VLANs 10, 20, 30, 40, 50, 60 y 70, routing inter-VLAN, ACLs de seguridad y aislamiento para red de invitados.

Sistemas y servicios

Integración de Windows Server 2019, Windows 11 Pro, Active Directory, GPO, DNS, DHCP, FTP, IIS y acceso remoto, con servidores dedicados para servicios internos.

Base de datos

Modelo de datos para registrar departamentos, usuarios, VLANs, equipos, servicios, eventos, incidencias, clientes, planes de servicio y facturación.

Seguridad

Aplicación de un enfoque Zero Trust, restricción del movimiento lateral, control de accesos, supervisión de intentos remotos y registro de incidencias técnicas.

Nube y documentación

Documentación del módulo de nube y organización del proyecto por asignaturas para mostrar la relación entre hardware, sistemas, redes, base de datos y lenguaje de marcas.

Trabajo en equipo

Proyecto realizado junto a Hugo López y Sergio Valiente, con reparto de módulos, defensa técnica y repositorio compartido para centralizar entregas.
